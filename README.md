## 🚀 Overview

EduChain is a full-stack scholarship disbursement system where funds
are: - Locked upfront - Evaluated per semester - Released only if
academic conditions are met



## 🧠 Core Concept

GPA ≥ 7.0\
Attendance ≥ 75%\
Credits ≥ 18



## 🏗️ Architecture

Frontend → Backend → Smart Contract Logic



## 📂 Project Structure

├── backend.js\
├── scholarship_frontend.html\
├── test.rs\
├── public\
├── README.md



## ⚙️ Features

-   Conditional fund release\
-   Real-time dashboard\
-   Student tracking\
-   Transparent evaluation



## ▶️ Run

Backend: node backend.js

Frontend: Open scholarship_frontend.html
